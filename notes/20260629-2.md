# Markdownでの表の書き方

Markdownでは`|`と`-`で表を定義する。
```
| カラム1 | カラム2 |
| --- | --- |
| 値1 | 値2 |
| 値3 | 値4 |
```
| カラム1 | カラム2 |
| --- | --- |
| 値1 | 値2 |
| 値3 | 値4 |

上２行は必須らしく、消したらGitHubで正しく表示されなかった。

またハイフンは各箇所で1個以上あれば良いらしい。

```
| カラム1 | カラム2 |
| - | - |
| 値1 | 値2 |
| 値3 | 値4 |
```
| カラム1 | カラム2 |
| - | - |
| 値1 | 値2 |
| 値3 | 値4 |
